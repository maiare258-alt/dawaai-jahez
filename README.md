# دواء - سلمية

مشروع أولي (MVP) لتطبيق يعرض توفر الأدوية في صيدليات مدينة سلمية لحظياً.

## بنية المشروع

```
dawaa-salamiya/
├── backend/
│   ├── routes/
│   │   ├── medicines.js  البحث عن الأدوية
│   │   ├── pharmacies.js تسجيل ودخول الصيدليات
│   │   └── stock.js      تحديث حالة توفر الأدوية
│   ├── data/
│   │   └── db.json       كل بيانات التطبيق مخزّنة هنا (بدون قاعدة بيانات منفصلة)
│   ├── db.js              طبقة التخزين
│   ├── server.js          نقطة انطلاق الخادم
│   └── package.json
└── frontend/
    ├── index.html
    └── app.js
```

## لماذا بدون PostgreSQL؟

كانت النسخة الأولى تعتمد على PostgreSQL، لكن تحميله محظور جغرافياً من بعض الدول عبر موقع التوزيع الرسمي. لذلك تم استبداله بتخزين بسيط في ملف `data/db.json` يعمل فوراً بدون أي تحميل إضافي. لاحقاً، عند الحاجة لقاعدة بيانات حقيقية (مثلاً عند نشر التطبيق على الإنترنت لعدد كبير من المستخدمين)، يمكن استبدال ملف `db.js` بقاعدة بيانات سحابية دون تغيير باقي الكود.

## خطوات التشغيل (Windows)

### 1. تجهيز الخادم

افتح "موجه الأوامر" (cmd) وروح لمجلد المشروع:
```
cd المسار\الى\dawaa-salamiya\backend
```

ثم:
```
npm install
```

### 2. التشغيل

```
npm start
```

إذا ظهر:
```
الخادم يعمل على http://localhost:3000
```
فكل شيء تمام.

### 3. فتح التطبيق

افتح المتصفح على: `http://localhost:3000`

## كيف يعمل التطبيق

**واجهة المريض:** بحث عن اسم دواء (تجاري أو علمي)، وعرض فوري لأي صيدلية بسلمية يتوفر فيها.

**لوحة الصيدلي:** كل صيدلية تسجّل حساباً (اسم مستخدم وكلمة مرور)، وتحدّث حالة كل دواء (متوفر / غير متوفر) بضغطة واحدة.

## ملاحظة مهمة

بيانات التطبيق محفوظة بملف `backend/data/db.json` — لا تحذف هذا الملف أو تعدله يدوياً وأنت لا تعرف صيغة JSON، لأن أي خطأ بصيغته سيوقف الخادم عن العمل. لعمل نسخة احتياطية، ببساطة انسخ هذا الملف لمكان آخر بين فترة وأخرى.

## الخطوات التالية المقترحة

1. جرّب التسجيل بصيدلية وهمية من "لوحة الصيدلي"، حدّث حالة بعض الأدوية، وابحث عنها من "واجهة المريض" للتأكد أن كل شيء متصل.
2. استبدل قائمة الأدوية التجريبية بقائمة حقيقية شاملة لأدوية السوق السوري.
3. سجّل صيدليات سلمية الفعلية بعد الاتفاق معها.
4. عند التوسع لعدد كبير من المستخدمين، يمكن الانتقال لقاعدة بيانات سحابية والنشر على الإنترنت.
